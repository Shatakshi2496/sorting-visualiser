# Sorting Algorithm Visualizer

An interactive web-based visualizer for classic sorting algorithms inspired by Visualgo.

## Features
- Bubble Sort, Selection Sort, Insertion Sort
- Dynamic bar count control
- Color-coded animations
- Slide-in algorithm explanation panel
- Dark and light mode

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
Open `index.html` in any modern web browser.
